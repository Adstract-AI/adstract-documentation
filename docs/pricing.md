Adstract SDK usage is completely free.

## Current price

- **SDK + API usage cost:** `0` (free)

## What this means for publishers

- You do not pay to use the SDK or API.
- Adstract is designed so publishers use the platform to earn from ad-enabled
  LLM experiences.
- The integration is a monetization path, not a usage-fee product.

## Next step

To start as a publisher:

- [Signup](/signup) and choose the `Publisher` role.

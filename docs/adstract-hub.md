Adstract Hub is a hosted conversational AI environment operated by Adstract.
It provides a unified interface where anyone can interact with multiple large
language models in one place — and it is completely free to use.

## What makes Hub different

Most products require you to implement the Adstract SDK before you can see
anything. Hub removes that requirement entirely.

Adstract is natively integrated into Hub, which means sponsored content flows
through every conversation the same way it would in a fully integrated
third-party application. You do not need to write any code to observe it in
action.

## Who it is for

- **End users** — interact with AI models for free and observe how contextual
  sponsored recommendations appear within responses, exactly as they would in
  any Adstract-integrated product.

- **Advertisers** — see how promotional content appears within real
  conversational contexts before committing to a campaign.

- **Publishers** — test the platform before integrating the SDK into your own
  application. Use Hub to observe behavior, verify your understanding of the
  enhancement flow, and build confidence before writing integration code.

## Testing with your own API key

Adstract Hub provides a testing interface that accepts a Publisher API key.
Enter your Sandbox or Billing key and Hub will route requests through your
platform configuration — letting you observe the full enhancement and
sponsored content flow without any implementation on your end.

This makes Hub the fastest way to confirm that your platform is configured
correctly — including all the parameters available on your platform setup in
Adstract — without deploying any integration code.

## Hub as a reference implementation

Hub serves as the reference implementation for the Adstract platform. The
behavior you observe there — how enhancements are injected, how sponsored
recommendations are presented, how the conversation flow is preserved — is
exactly what your own integration will produce once the SDK is in place.

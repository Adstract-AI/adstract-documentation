title: Signup

---

Use this guide to create an Adstract account and select the correct role.

## Step 1: Open signup

1. Go to [adstract.ai](https://adstract.ai/).
2. Click `Login / Signup` in the top header navigation.
3. Click `Sign up`.

## Step 2: Complete account fields

On the signup form, enter:

- `First Name`
- `Last Name`
- `Email` (must be valid)
- `Password`
- `Confirm Password`

## Step 3: Complete profile panel

In the left panel, enter:

- `Display Name`
- `Role`
  - `User`
  - `Publisher`
  - `Advertiser` (coming soon)

For API key access, select **Publisher**.

You can also optionally add:

- `Picture URL` (optional profile image link)

## Step 4: Create account

After all required fields are complete, click `Create Account`.

## Role guidance

If you are unsure which role to pick, review the role pages:

- [User](/role-user)
- [Publisher](/role-publisher)
- [Advertiser](/role-advertiser)

These pages explain ownership model and expected use for each role.
